# Cloven CLI Tool

A Simple Bridge between the web and your projects

## Installation

Install cloven using npm

```bash
  npm install -g cloven
```

## Acknowledgements

- [Nodedactly](https://github.com/Nodeactyl)

## Deployment

To upload a project to a server you own run

```bash
  cloven upload
```
